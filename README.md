## Little Troubles Cookie Counter

This is a cookie & fight tracker for the game Little Troubles.

### Hardware

- ESP32-based chips
- SPI Displays
- I2C Touchscreen

#### Dev Kit

- Dev Board: ESP32-2424S012C-I
- CPU: ESP32-C3
- Display: 1.2" SPI display (240x240 pixels, GC9A01 driver)
- Touchscreen driver: CST816D
